# 📚 Library Management System

A simple and functional web application to manage your personal book collection. This project focuses on CRUD (Create, Read, Update, Delete) operations using vanilla JavaScript.

## 🚀 Features

* **Book Entry Form:** A clean interface to input book details, including:
    * **Book Name** (Title)
    * **Author**
    * **Page Count**
    * **Read Status:** A checkbox to indicate if you have already finished the book.
* **Search Bar:** Quickly find specific titles or authors in your collection with a real-time search filter.
* **Status Toggle:** A dedicated button on each book card to mark it as "Read" or "Not Read" without deleting it.
* **Remove Tool:** A delete button to instantly remove a book from the library list.

---

## 🛠️ Technologies Used

* **HTML5:** Semantic structure for the web pages.
* **CSS3:** Custom styling for the layout and book cards.
* **JavaScript (ES6+):** Logic for handling book objects, search functionality, and DOM manipulation.

---

## ⚠️ Important Note

> **Note on Responsiveness:** This project is currently **not responsive**. It was designed and optimized specifically for desktop browsers. Mobile support may be added in future updates.
